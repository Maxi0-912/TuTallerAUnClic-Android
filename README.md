# 🚗 TuTallerAUnClic - Android App

Aplicación móvil desarrollada en Kotlin + Jetpack Compose
para gestión de citas en talleres y lavaderos.

## 🚀 Tecnologías
- Kotlin
- Jetpack Compose
- Navigation Compose
- Retrofit
- JWT Authentication
- MVVM Architecture

## 🔐 Backend
Conectado a API Django REST con autenticación JWT.

## 📱 Funcionalidades actuales
- Login con JWT
- Persistencia de sesión
- Navegación Compose
- Arquitectura limpia (MVVM)
